Hackers Breach npm Developer Account and Inject Malicious Code into 18 Popular Packages with Over 2 Billion Weekly Downloads

Author: Juan Pablo Dueñas · Source: Fluid Attacks

A recent supply chain attack on npm compromised 18 popular JavaScript packages with more than 2 billion weekly downloads. The incident began when attackers gained access to the account of Josh Junon (known online as Qix), a well known open source developer, through a sophisticated phishing email. Once inside, they uploaded malicious versions of several widely used packages.

The injected code was designed to intercept and modify cryptocurrency transactions made from web browsers, secretly replacing wallet addresses with the attacker’s own. While not every project using these libraries was directly affected, the potential impact was massive because many of the compromised packages are core dependencies used across thousands of applications and frameworks.

This incident highlights how fragile the open source ecosystem can be when security depends on just a few maintainers. It also reinforces the need for stronger protection measures such as multi factor authentication, dependency integrity checks, and security scanning tools to prevent a single compromised account from putting millions of users at risk.
